Use the tinyfish web agent when the task involves:
- scraping UI inspiration websites (e.g. cosmos.so, dribbble, awwwards)
- extracting layout structures, color palettes, typography, and spacing
- analyzing component hierarchy and interaction patterns

When scraping design inspiration:
1. Identify page sections (hero, navbar, cards, grids, footers)
2. Extract:
   - layout structure (flex/grid)
   - spacing scale
   - font styles and hierarchy
   - color palette
   - animation patterns (hover, transitions)
3. Convert findings into:
   - clean React + Tailwind components
   - reusable UI sections
   - modern design patterns (glassmorphism, gradients, minimalism)

Always:
- prefer clean, minimal, modern UI
- use proper spacing and typography hierarchy
- ensure components are production-ready
